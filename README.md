# OpenAPI Go Codegen CLI

A command-line tool that generates idiomatic Go code (models, clients, and optionally server stubs) from OpenAPI 3.x specifications.

## Features

- Generate Go models from schemas
- Generate HTTP client code from API operations
- Optional server interface/stubs
- Works with local files or remote URLs
- Supports YAML and JSON OpenAPI specs
- Customizable package name and output layout
- Config file support for repeatable runs
- Helpful diagnostics and validation
